# T7Capping_2024
Git repository for preprint Buchanan et al 2026, that describes how the inclusion of helper plasmids encoding T7 RNA polymerase, and Vaccinia virus capping enzymes D1 and D12 permits rescue of calicivirus reverse genetics plasmids that express the viral genome from a T7 promoter and incorporate a 3' poly(A) tail and HDV ribozyme.
